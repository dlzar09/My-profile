## 🧠 About the Portfolio Concept

This is not a normal static portfolio.  
The design uses a **“Signal System”** idea, where the page feels like a smart technical map coming online as the visitor scrolls.

The goal is to quickly communicate:

- technical ability
- systems thinking
- leadership mindset
- real business experience
- continuous learning
- professional creativity
